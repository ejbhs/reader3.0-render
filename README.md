# Reader 3.0 On Render
A repo that deploys the infamous Chinese Webnovel app "Reader 3.0" on render.com for free. This deployment comes prebuilt with a comprehensive set of Shuyuan (书源) sources that cover over 2000+ book sites, enabling seamless access to a vast library of novels without additional configuration.

## Deploy

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy)
## Features

- **Free Hosting**: Deploy on Render.com without cost
- **2000+ Book Sources**: Comprehensive Shuyuan (书源) collection pre-configured
- **No Additional Setup**: Works out-of-the-box with extensive novel site coverage
- **Reader 3.0 App**: Full functionality of the popular Chinese webnovel reader

## Project Goals

This project aims to:
- Provide free, accessible deployment of Reader 3.0 for the global community
- Eliminate configuration barriers by including a curated library of book sources
- Enable seamless access to Chinese webnovels and light novels
- Maintain an up-to-date collection of working novel sources

## Getting Started

### Prerequisites
- A Render.com account (free tier available)

### Deployment Steps

1. Click the "Deploy to Render" button above
2. Follow the Render deployment wizard
3. The app deploys with pre-configured sources
4. Access your instance once deployment completes

## Usage

Once deployed, access your Reader 3.0 instance through the provided Render URL. The app includes:
- Built-in search across 2000+ novel sources
- Reading history and bookmarks
- Custom source management
- Dark/light theme options
